  • dc:description "The Graphical Output Ontology provides a description of graphical climate products (charts, maps...), including a description of uncertainty communication. It has two main components: i) The graphical product description and ii) the description of the uncertainty types communicated by the different graphical elements of the product. There are specific concepts to describe the types of uncertainty and how this is communicated within the product. It extends the ds:Step class from METACLIP's Data Source Ontology introducing the generation of the graphical product as an additional step within the climate data workflow." @en
